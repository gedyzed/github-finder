## Getting Started

1. **Clone the repo**
```
git clone https://github.com/gedyzed/github-finder.git
cd github-finder
```

2. Install dependencies
```
npm install
```
3. Create an .env file
4. Add your GitHub Token to an .env
5. start server
```
npm run dev
```
